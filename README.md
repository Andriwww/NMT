# Network Monitoring Tool

## 📌 Descripción
Herramienta en Python para monitorizar la disponibilidad y latencia de dispositivos en una red mediante ping.

## ⚙️ Funcionalidades
- Comprobación de estado (UP/DOWN)
- Medición de latencia
- Ejecución periódica

## 🛠️ Tecnologías
- Python
- subprocess
- ICMP (ping)

## 🚀 Cómo ejecutar
```bash
python main.py
